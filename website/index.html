The Swiss Transportation System
This histogram shows the distribution of different transport types used throughout the day, divided into 30-minute intervals. Users can interact with the graph by selecting the transport type they want to visualize using the buttons provided. The available transport types include Train, Bus, Boat, Metro, Tram, and Rack Railway. The height of each bar represents the frequency of that transport type during the specified time interval. This visualization allows you to compare the usage patterns of different transport types and identify peak usage times.
This visualization displays a Hierarchical Edge Bundling of all the train stops in Switzerland, organized by canton. Users can select one or multiple cantons to display on the graph, allowing for easy comparison of train stop connections between different regions. The thickness of each edge represents the number of passengers traveling between the two stops, while the color indicates the average travel time between stops. This interactive visualization provides a comprehensive overview of Switzerland's train transportation network and highlights the connections between different regions.
This Circle Packing plot displays the different transport types that exist in Switzerland and the lines that belong to each type. The size of each circle represents the number of entries on that line. Users can interact with the graph by clicking on one of the circles to focus on it and see the lines. This visualization provides an intuitive and informative overview of Switzerland's transportation system, allowing users to easily explore and understand the different transport types and lines available.
